Spiegazione dei servizi di stampa

  • xConvert
    Parametri Input:
    1. mode : Modalità di restituzione del risultato (show/other)
    2. filename : Nome del file di uscita
    3. docurl : Url del file da convertire
    4. file : file codificato in base 64 da convertire
    5. TODO data : dizionario con valori da sostituire prima della conversione
    Output: Se mode è "show" restituisce il binario del file convertito con header "application/pdf"
    Altrimenti restituisce una json composta da:
    1. success : (0/1) se il processo è andato a buon fine
    2. file : file codificato in base 64 convertito in pdf (se success è 1)
    3. message : messaggio di errore (se success è 0)
  • xCreate
    Parametri Input:
    1. mode : Modalità di restituzione del risultato (show/other)
    2. filename : Nome del file di uscita
    3. model : Nome del modello di stampa
    4. project : nome del progetto
    5. app : nome dell'applicazione
    6. group : sottocartella dell'applicazione contenente i modelli di stampa
    7. id : id dell'oggetto chiamante
    8. TODO file : Template file codificato in base 64
    9. data : dizionario con valori da sostituire
    Output: Se mode è "show" restituisce il binario del file convertito con header "wordxml"
    Altrimenti restituisce una json composta da:
    1. success : (0/1) se il processo è andato a buon fine
    2. file : file codificato in base 64 convertito in docx (se success è 1)
    3. message : messaggio di errore (se success è 0)
